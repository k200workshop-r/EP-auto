# EP 課程報表整理網站

這個小網站可上傳原始 Excel 報表與教師姓名對照表，套用指定欄位邏輯後下載整理後的 Excel。

## 處理邏輯

- 新增「排除重複課程」欄位，不覆蓋原始 H 欄；內容以 I 欄 + Q 欄 + S 欄組合，中間用 `|` 分隔，並依此移除重複資料。
- 依教師姓名對照表比對授課老師與協同老師是否有員編，取代原本 VLOOKUP。
- AD 欄命名為「課程分類」。
- 若 T 欄為「授課老師有員編」，「課程分類」填入「A.住院醫師主授課程」。
- 若 V 欄為「協同老師有員編」，「課程分類」填入「B.住院醫師擔任協同老師」。

## 啟動方式

```powershell
pip install -r requirements.txt
python app.py
```

啟動後開啟：

```text
http://127.0.0.1:5000
```

## 教師對照表欄位

教師對照表需包含姓名與員編欄位。欄名可使用「姓名」「教師姓名」「老師姓名」及「員編」「員工編號」「教師員編」等常見名稱。
